> v0.0.7 集中优化悬浮窗操作：扩大可拖动区域、固定窗口控件位置，并移除最外侧黑色细线。

## Highlights

- **更多位置可以拖动**：左侧九点拖动柄、顶部品牌区域、列表剩余空白区域和底部空白区域均可拖动悬浮窗。
- **切换尺寸不再晃动**：迷你模式和展开模式共用同一套 `48px` 高固定工具栏，九点拖动柄与右侧三个按钮保持相同大小和位置。
- **边缘更加干净**：关闭悬浮窗原生缩放能力、原生阴影及 CSS 外框，移除 Windows 无标题栏窗口外侧的黑色细线。

## 变更

- 悬浮窗继续固定为 `420px` 宽度；迷你和展开仅改变高度。
- 窗口控制区固定为三个 `30px` 槽位，避免标题和事项内容影响图标位置。

## 修复

- 修复只有顶部部分区域能够可靠拖动悬浮窗的问题。
- 修复放大或缩小时左侧九点和右侧三个窗口图标发生位移或尺寸变化的问题。
- 修复悬浮窗最外侧仍显示 Windows 原生黑色细线边框的问题。

> [!WARNING]
> 安装 v0.0.7 前请从托盘正常退出旧版。升级不会主动删除 `%APPDATA%\in-line` 中的用户数据库。安装包目前未使用商业代码签名证书，Windows SmartScreen 可能提示未知发布者。
