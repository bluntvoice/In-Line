> v0.0.3 是面向 Windows 的紧急稳定性修复版，解决安装后黑色控制台、主界面持续“整理中”和悬浮窗操作失效问题。

## Highlights

- **启动恢复正常**：发布构建不再显示黑色命令行窗口，数据库初始化异常会显示错误和重试入口。
- **悬浮窗可用**：新增取号、查看、编辑、修改状态和加急操作会正确打开并定位目标事项。
- **队列反馈完善**：修复事件监听竞态，上移或下移失败时给出明确提示。
- **数据保持兼容**：继续使用 `%APPDATA%\in-line\inline.db`，不清空事项、号码、日志或备份。

## 修复

- 修复 Windows GUI 发布包附带控制台窗口的问题。
- 修复数据库初始化失败时主界面无限停留在“正在整理队列”的问题。
- 修复悬浮窗“新增取号”无法可靠打开新增表单的问题。
- 修复悬浮窗右键操作未定位对应事项的问题。
- 修复前端事件监听销毁竞态和队列移动失败没有提示的问题。

> [!WARNING]
> 如果“设置 → 应用”仍显示 Electron 旧版 In Line 0.0.1，请先正常退出并卸载旧版，再安装 v0.0.3。卸载和升级不会主动删除 `%APPDATA%\in-line` 中的用户数据库。安装包目前未使用商业代码签名证书，Windows SmartScreen 可能提示未知发布者。
