> 首个可日常使用的稳定重构版本：更轻量、更清晰，并保留旧版事项与编号数据。

## Highlights

- **更小的安装体积**：桌面架构迁移到 Tauri 2，Windows x64 NSIS 安装包约 2 MB。
- **人工顺位优先**：每个事项可直接上移或下移，重启后顺序保持不变。
- **单击复制取号卡片**：主列表和悬浮窗共用同一复制服务，避免重复生成。
- **数据安全升级**：沿用旧 SQLite 数据，迁移前自动备份，并支持安全恢复。
- **通用工具定位**：界面与默认事项类型不再限定法务场景。

## 新增

- 结构化状态历史与加急历史。
- 备份列表、恢复入口、系统级新增快捷键和单实例运行。
- 透明应用图标以及统一的复制、上移、下移操作图标。

## 变更

- Electron 运行时替换为 Tauri 2、Rust 与系统 WebView2。
- 队列排序改为人工顺位绝对优先；加急、优先级和逾期只作醒目标识。
- README 使用项目方提供的原始 PNG 品牌图。

## 修复

- 修复 Windows 125%–200% 缩放下字体错位、中文基线偏移和内容裁切。
- 修复复制图片入口重复、透明悬浮区域挡鼠标及本地日期边界错误。
- 修复窗口、托盘、安装包和快捷方式未正确使用应用图标的问题。

> [!WARNING]
> 安装包目前未使用商业代码签名证书。Windows SmartScreen 可能显示未知发布者提示，请确认文件来自本仓库后再安装。
